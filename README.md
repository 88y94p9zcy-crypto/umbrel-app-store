# 88y94p9zcy-crypto Umbrel App Store

Osobní Umbrel App Store s crypto nástroji.

## Instalace na Umbrel

1. Otevři Umbrel dashboard
2. Jdi do **App Store**
3. Klikni na ikonu nastavení (⚙️) vpravo nahoře
4. Klikni **Add community app store**
5. Vlož tento odkaz:
   ```
   https://github.com/88y94p9zcy-crypto/umbrel-app-store
   ```
6. Klikni **Add** — appky se zobrazí v App Store

## Dostupné appky

### BTC DCA Bot
Automatický DCA bot pro Binance s dip buyingem a Telegram notifikacemi.
- 4 úrovně dip buyingu
- Telegram příkazy (/stav, /koupit, /zustatek)
- Sledování zůstatku s upozorněním
